# Movieflix

Movieflix is a modern, responsive web application inspired by popular streaming platforms. It allows users to browse movies by category, watch local videos, search globally, and manage user authentication (signup/login). This project is built with **HTML, CSS, and JavaScript**, providing a fully interactive and dynamic experience.

---

## üîπ Features

- Browse movies by categories: **Comedy, Action, Romance, Anime, Family, Horror, Real, Series, 90s, Premium**
- Responsive **interface page** with hero section and signup prompt
- **Login and Signup** functionality with form validation
- **Search bar** with live filtering of movies across all categories
- **Movie Grid** layout with hover animations
- **Video Modal** for playing local videos
- **FAQ section** with expandable questions
- Fully responsive **footer** and layout for all screen sizes
- Persistent user storage using **localStorage**
